# Garbage

`garbage/` stores retired noise, invalidated artifacts, and superseded verbose history.

Use this directory when the raw material no longer deserves a place in the active context but should still remain traceable.

## Rules

1. Do not move the only copy of a meaningful decision here.
2. Preserve the lesson in active memory before retiring raw failure detail.
3. Add every retirement to `garbage/index.json`.
4. Use repository-relative paths only.

## Index expectations

Each garbage record should capture, when applicable:

- why the material was retired,
- which lesson was preserved,
- what replaced it,
- whether it can be restored.
